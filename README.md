# Flyer Tools

我的Chrome工具箱扩展程序

## 功能清单

### 快捷搜索
  
+ 可自定义搜索引擎的右键快捷搜索菜单

+ 支持对网页中选中的文本进行多搜索引擎搜索

+ 支持定制搜索标签的打开方式

+ <kbd>HOME</kbd> 和 <kbd>END</kbd> 快捷键快捷搜索

### 谷歌翻译

+ 谷歌翻译网页时禁止翻译代码展示元素

+ 支持自定义代码容器元素CSS选择器

### YouTube

+ 添加YouTube视频播放器网页全屏功能
